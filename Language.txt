Modes

@CV - Switch to CV Mode
@MIDI - Switch to MIDI Mode
@EM - Switch to CE-126P Terminal Emulation

@CV Commands

+   Gate On
-   Gate Off
n   Emit voltage
    n is a floating-point number between 0 and 5
    
@MIDI Commands

Cn  Set MIDI channel
Vn  Set MIDI velocity
n   Send MIDI note on
X   Send MIDI note off
